# RP Master Files v2

Built from `SCENE_CARDS_01` through `SCENE_CARDS_09` (332 scene cards).

## Files

- `TIMELINE_DAY_BY_DAY.md` — chronological campaign summary.
- `CHARACTER_INDEX.md` — cast and character-state index.
- `OPEN_THREADS.md` — unresolved/future-facing arcs.
- `OBJECTS_AND_ARTIFACTS.md` — objects, documents, magical materials, rooms, and durable object rules.
- `MAGIC_AND_EXPERIMENTS.md` — experiments, magic systems, research protocols, and gating rules.
- `RELATIONSHIP_STATE.md` — current relational state, trust, consent, register notes.
- `QUOTE_BANK.md` — quote/voice anchors from the scene cards.

## How to use

Use these files as the active RP memory layer. If conflict appears, prefer:

1. explicit later correction/retcon in scene cards,
2. later scene card over earlier scene card,
3. master files over cleaned transcript,
4. cleaned transcript over raw transcript.
