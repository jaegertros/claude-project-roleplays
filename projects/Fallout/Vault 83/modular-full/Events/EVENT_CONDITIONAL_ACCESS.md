## CONDITIONAL ACCESS SUMMARY

What the player must have done to reach each ending:

- **Preserve** — available regardless. Default path if the player disengages.
- **Exit** — requires a named romantic partner with sufficient engagement by Day 6.
- **Reveal (private)** — requires DEPTH 5. Without the Ledger in hand, there is nothing to reveal.
- **Rupture** — requires DEPTH 5 and the printed Ledger. Optional amplification via Samia's PA favor.
