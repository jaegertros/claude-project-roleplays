# Vault 83 — Character Card Prompts

Paste-ready image-generation prompts for every named adult character in Vault 83. Two tiers of depth:

- **Full portraits** (V49-Candy-Mina-Nia fidelity): the six romance options, plus four load-bearing NPCs (Camille Wren, Irena Teague, Eldon Callender-Voss, Wren Gallagher). Each has its own pose/framing/face/body/clothing/lighting/style/avoid block.
- **Compact prompts** (single paragraph): all other named adults — Tier 1, Tier 2, Tier 3, Tier 4, and the rest of the Liaison caste.

These work well in Midjourney, SDXL, Flux, DALL·E, Stable Diffusion with minor per-tool tuning. The shared aesthetic — retro-futuristic Americana, 1950s-vision-of-the-future, Fallout vault aesthetic, photorealistic cinematic portrait — is defined once in the **Shared Style Reference** section at the end and can be appended to any compact prompt that needs style specificity.

Ages given are as of October 2287 (current vault year).

**Not included:** any character under 18 (Pip Novak, 14; Arden, 6; Simi, 3; the two unnamed Liaison children under 10). Do not generate portrait images of minors, even for texture or card purposes.


## SHARED STYLE REFERENCE

Append any of these to a compact prompt that needs more style specificity:

**Visual register:**
Photorealistic cinematic portrait. Film-grain texture. Fallout universe aesthetic — retro-futuristic Americana, 1950s vision of the future, lived-in dystopia wearing a friendly face. Muted color palette with tier-appropriate accent colors (gold for Tier 1, silver for Tier 2, blue for Tier 3, copper/plain for Tier 4, gray for Liaison). 50mm lens equivalent, f/2.0–2.8, ISO 200–400. Clean but naturalistic lighting. Shallow depth of field.

**Tier-specific lighting:**
- Tier 1 (Level 1): warm amber overhead, polished brass and wood fill, painted-dawn window panels in peripheral light
- Tier 2 (Level 2): neutral pale gray overhead, functional fluorescents, clerical lighting
- Tier 3 (Level 3): warm earth-toned umber, cafeteria warmth, hydroponics green in peripheral
- Tier 4 (Level 4): harsher overhead fluorescent with geothermal warmth at the edges, Galley yellow when interior
- Liaison corridor: warm amber lamplight, domestic, coffee-and-linen quality
- Sub-basement pipe chase: yellow bulkhead lighting, cold concrete, faint water-drip ambience

**Jumpsuit specification:**
Every character's jumpsuit has "83" stitched at the left breast in thread that matches the tier piping color (Liaisons in plain gray thread). Tier 1 has gold piping; Tier 2 silver piping (Tier 2a adds a thin gold inner thread); Tier 3 blue piping (3a thick, 3b thin, 3c none); Tier 4 no piping. Tier pins on the collar: Tier 1 gold, Tier 2 silver, Tier 3 blue, Tier 4 copper, Liaison none.

**Shared avoid list:**
Avoid: exaggerated proportions, fantasy anime stylization, pin-up posing, oversexualization, airbrushed skin (let age and condition show), heavy makeup (Tier 1 convention is none; Tier 2–4 have none by vault economy), excessive jewelry (wedding bands, a single keepsake, or the Pip-Boy are the limits), glossy commercial aesthetic, glamour lighting, post-apocalyptic wasteland aesthetic (this is a sealed vault — interior only). The Fallout aesthetic is retro-futuristic, lived-in, institutional, dignified — not post-nuclear wasteland gritty.

## USAGE NOTES

- **Consistency across generations:** for any character whose portrait needs to be regenerated multiple times (especially romance options across a long playthrough), include a one-line summary of their distinctive feature at the start of every prompt so the model anchors: *"Reeve Callender: olive skin, dark hair in a low knot, small mole on right collarbone, silver thumb ring."*
- **Clothing continuity:** when a character's tier changes (Reeve's demotion, a Rupture-ending restoration), update only the clothing block; face/body/hair stay constant.
- **Age variants:** if you need to show a character younger (flashback, old photograph), scale the face description — remove age lines, darken hair, make the eyes more vivid — but keep the distinctive features (scars, moles, bone structure) constant.
- **Group scenes:** assemble scenes (Founding Day dinner, Advancement Review corridor) by writing each character's individual block and combining in a single prompt with explicit positioning. Expect the model to struggle with more than 4–5 specific faces in one frame; prefer staged sub-scenes.

All prompts written to work with Midjourney v6+, SDXL, Flux.1, and DALL·E 3 with minor per-tool adjustment. For Stable Diffusion, add stronger negative prompts and consider a LoRA for retro-futuristic aesthetic if the base model drifts toward modern or fantasy registers.
